🤖 Chatbot Widget Demo

A modern, standalone chatbot widget that works with any framework

Angular 9+ React Vue Vanilla JS

✨ Demo 1: Floating Widget

Look at the bottom-right corner of this page! Click the purple button to open the chat.

<!-- Just add this to your HTML --> <script src="chatbot-widget.min.js"></script> <chatbot-widget api-url="http://localhost:5112" title="AI Assistant" floating="true"> </chatbot-widget>
Note: The floating widget appears in the bottom-right corner and stays visible as you scroll.

📱 Demo 2: Inline/Embedded Widget

Embed the chatbot directly into your page layout:

<div style="height: 600px;"> <chatbot-widget api-url="http://localhost:5112" title="Help Desk" floating="false"> </chatbot-widget> </div>

⚙️ Using in Angular 9

Perfect for legacy Angular applications! Here's how to integrate:

Step 1: Add the script to angular.json

{ "projects": { "your-app": { "architect": { "build": { "options": { "scripts": [ "src/assets/js/chatbot-widget.min.js" ] } } } } } }

Step 2: Enable custom elements in app.module.ts

import { NgModule, CUSTOM_ELEMENTS_SCHEMA } from '@angular/core'; @NgModule({ // ... other config schemas: [CUSTOM_ELEMENTS_SCHEMA] }) export class AppModule { }

Step 3: Use in any template

<!-- app.component.html --> <div class="app"> <h1>My Angular 9 App</h1> <router-outlet></router-outlet> <chatbot-widget api-url="http://localhost:5112" title="Support"> </chatbot-widget> </div>
